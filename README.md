# capsuru
情報表現基礎Ⅱの未来大周辺に生息するデジタル仮想生物を作成するという課題での成果物.

![capsuruの写真](capsuru.jpg)

# Requirement
* Arduino IDE
* Arduino Leonardo
* [VarSpeedServo](https://github.com/netlabtoolkit/VarSpeedServo)
